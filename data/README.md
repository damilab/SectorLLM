# Data

| File | Description | Source |
|------|-------------|--------|
| `llm_questions_2012_2021.csv` | LLM input prompts with firm characteristics | WRDS (CRSP + Compustat) |
| `question_features_2011_2021.csv` | Features table for merging model outputs | WRDS (CRSP + Compustat) |
| `prices_2012_2021.csv` | Daily stock prices for backtests | Yahoo Finance |
| `permno_monthly_meta.csv` | Monthly metadata (price, return, shares outstanding, etc.) | WRDS CRSP |

Firm characteristics are from WRDS (CRSP + Compustat), stock prices are from Yahoo Finance.
